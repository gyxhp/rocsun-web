### React 项目目录结构
```text
my-react-app/
  ├─ public/               # 静态资源（favicon  等）
  ├─ src/
  │  ├─ assets/            # 图片、样式
  │  ├─ components/        # 可复用组件
  │  ├─ App.jsx            # 根组件
  │  ├─ main.jsx           # 入口文件
  ├─ index.html            # 页面入口
  ├─ package.json
  └─ vite.config.js        # Vite 配置文件
```
